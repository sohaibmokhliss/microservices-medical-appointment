# Healthcare Frontend Application

Application React pour le système de prise de rendez-vous médicaux.

## Installation

```bash
npm install
```

## Démarrage

```bash
npm start
```

L'application sera disponible sur http://localhost:3000

## Prérequis

Les services backend doivent être démarrés:
- docteur-service sur le port 8081
- rdv-service sur le port 8082
- notification-service sur le port 8083

## Fonctionnalités

- Affichage de la liste des docteurs
- Création de rendez-vous
- Affichage des rendez-vous
- Envoi de notifications par email
